# Tic Tac Toe

A simple Tic Tac Toe game built by recreating the logic of a C console program using **HTML, CSS, and JavaScript**.

This project was created to strengthen my understanding of programming logic, arrays, functions, and game development while also learning basic frontend web development.

---

## Features

- Two-player game (Player X and Player O)
- Automatic win detection
- Draw detection
- Restart game button
- Clean black-and-white user interface
- Original C source code included

---

## Technologies Used

- C (Game Logic)
- HTML5

##  How to Run

### Option 1 (Recommended)

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html**.

OR

Open `index.html` using **Live Server** in VS Code.

---

### Running the C Version

Compile using GCC:

```bash
gcc tic_tac_toe.c -o game
```

Run:

```bash
./game
```

On Windows PowerShell:

```powershell
.\game.exe
```

---

## How to Play

1. Player **X** makes the first move.
2. Players take turns selecting an empty square.
3. The first player to complete:
   - a row
   - a column
   - or a diagonal

wins the game.

If all nine squares are filled without a winner, the game ends in a draw.
